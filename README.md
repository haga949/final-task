# アニメ情報サイト

## 変更点
1. ホームページ(index.html)
- 放送中・おすすめ・ランキングなどのアニメを一覧で閲覧できるWebサイト

・各作品一覧を横スクロール形式に変更し、ドラッグ・スワイプ操作に対応
・広告欄を追加（現在Sample画像）、スマホサイズにしたとき各項目の間に広告欄が変更されるようにした

2. 作品一覧(anime-list.html)
・ジャンルをそれぞれボタンに変更
・アニメ一覧の見せ方をアニメの一部をスクロール形式に変更

3. ランキング(anime-ranking.html)
・TOP3位を4位以下のカードよりも大きくし、強調。
・縦に一列並ばせるのでは無く、3位以上をトップに配列（PCサイズの場合に限る）させ、4位以下を横並びカード形式に変更

4. 作品詳細(anime-detail.html)（ホームページの「あなたにおすすめのアニメ」欄の「ドラゴンボールZ」をクリックすることで遷移）
・画像を左に1枚、右に2枚というレイアウトに変更
・エピソード欄の追加
・「キャラクター」　→　「シリーズ/関連のアニメ」に変更

5. ログインページ(anime-login.html)
・  ログインヘッダーの変更
